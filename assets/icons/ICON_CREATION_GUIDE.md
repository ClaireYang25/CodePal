# Gmail OTP AutoFill - 图标创建指南

## 🎨 创建扩展图标

由于 Chrome Web Store 需要特定格式的图标文件，您需要创建以下图标：

### 需要的图标尺寸：
- `icon16.png` - 16x16 像素
- `icon48.png` - 48x48 像素  
- `icon128.png` - 128x128 像素

### 快速解决方案：

#### 方法 1: 使用在线图标生成器
1. 访问：https://www.favicon-generator.org/
2. 上传一个简单的 OTP 相关图片
3. 下载生成的图标包
4. 重命名文件为 `icon16.png`, `icon48.png`, `icon128.png`
5. 放入 `icons/` 文件夹

#### 方法 2: 使用 Canva 或 Figma
1. 创建 128x128 像素的画布
2. 设计一个简单的 OTP 图标（例如：数字"123456"或"OTP"文字）
3. 导出为 PNG 格式
4. 使用图片编辑软件调整尺寸

#### 方法 3: 使用简单的文字图标
创建一个简单的蓝色背景 + 白色文字"OTP"的图标

### 图标设计要求：
- 背景色：#4285F4 (Google 蓝色)
- 文字色：白色
- 字体：Arial 或类似的无衬线字体
- 简洁明了，易于识别

### 临时解决方案：
如果急需上传，可以先使用纯色方块作为临时图标，后续再更新。
